# Foundational Repo

## What this repo is for
This repository serves as the **default starting kit for projects**. It provides a structured foundation for building robust, scalable research and development workflows, ensuring consistency across all new initiatives.

## How to use it
You can use this repository in two ways:
1.  **Template Repository**: Click "Use this template" on GitHub to create a new repository based on this structure.
2.  **Copy**: Clone or download this repository and copy the contents into your new project directory.

## Folder Map
*   `AGENTS.md`: The entrypoint for AI agents, describing the repo structure and rules.
*   `/S00-foundation/`: Core system configurations and foundational documents.
*   `/S10-primitives/`: Basic building blocks of the system, including definitions for Agents and Skills.
*   `/S20-workflows/`: Documented procedures and workflows for executing tasks.
*   `/S30-examples/`: Reference implementations, including `example-research-brief.md`.
*   `/S90-assets/`: Supporting assets, clip maps, and graphics specifications for the bonus lesson.

## The Operating Loop
The core operating loop of this system follows a precise sequence: gather **Context** to inform the strategy, develop a detailed **Plan**, execute the **Run** to produce work, **Review** the outputs against established criteria, and **Bank** the learnings and valid artifacts back into the repository for future leverage.

## Quickstart Checklist
*   [ ] Clone or use this repository as a template for your new project.
*   [ ] Read `/10-primitives/agents.md` to understand the available agent roles and permissions.
*   [ ] Review `/10-primitives/skills.md` to understand how to define and use skills.
*   [ ] Select an initial workflow from `/20-workflows/` (e.g., `workflow-context-to-output.md`).
*   [ ] Execute your first run following the "Context -> Plan -> Run -> Review -> Bank" loop.
